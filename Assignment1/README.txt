WHAT IS IT?
===========
This folder contains the code for CS561 Assignment 1 by group 23.
This has the implementation for 2 feature selection techniques : 1) Sequential Forward Selection
																 2) Hill Climbing Technique

HOW TO RUN?
===========
The code can be run on any dataset with 2 conditions:
	1. The data cannot have any missing values.
	2. The last column must be the Target variable

To run the code, update the path of the csv file in the beginning of the code and the separator. Then run it from terminal.
The code depicts the progress of the algorithm in a graph.
